## About Akan Names.
Akan names originate from the Ghanian Community.
They are based on the day the child is born in and whether teh child is female or male.

## Description of the project.
This website is built so that one could know what their name would be if they were born into the Akan tribe.

## Set-up Instruction.
All you have to do is input your details of birth and run the wbesite by clicking generate your Akan name.
